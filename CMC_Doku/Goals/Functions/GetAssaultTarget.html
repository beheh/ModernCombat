GetAssaultTarget

Kategorie: Ziele (global)

Beschreibung

Gibt ein Zielobjekt anhand seiner Teamzugehörigkeit und seiner bei AddAssaultTarget angegebenen Nummer zurück. Ist das Ziel bereits zerstört, wird 0 zurückgegeben.

Syntax

object GetAssaultTarget (int iIndex, int iTeam);

Parameter

iIndex:
Index des Zielobjekts.
iTeam:
Team, dessen Zielobjekt abgefragt werden soll.

Siehe auch

Objekte: Assault, Base Assault
Funktionen: AddAssaultTarget, ConnectAssaultTargets
Calls: OnAssaultTargetDestruction
