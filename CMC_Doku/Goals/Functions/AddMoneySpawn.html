AddMoneySpawn

Kategorie: Ziele (global)

Beschreibung

Erzeugt einen Geldsack für das Spielziel Money Run und gibt den erzeugten Spawnpunkt zurück.

Syntax

object AddMoneySpawn (int iX, int iY, any MoneyCount);

Parameter

iX:
X-Position des neuen Geldspawns.
iY:
Y-Position des neuen Geldspawns.
MoneyCount:
Geldmenge, die der Spawnpunkt besitzt. Wird ein Array angegeben, wird zufällig einer der Werte ausgesucht.
