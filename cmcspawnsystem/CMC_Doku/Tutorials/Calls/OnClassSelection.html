OnClassSelection

Kategorie: Calls

Beschreibung

Wird von der Klassenwahl aufgerufen, wenn ein Spieler eine Klasse ausgewählt hat.

Syntax

public func OnClassSelection (object pClonk, int iClass)

Parameter

pClonk:
Clonk, der die neue Klasse gewählt hat.
iClass:
Nummer der Klasse, die gewählt wurde.
