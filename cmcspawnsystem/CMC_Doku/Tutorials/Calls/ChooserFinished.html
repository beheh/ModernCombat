ChooserFinished

Kategorie: Calls

Beschreibung

Wird aufgerufen, wenn die Regelwahl abgeschlossen wurde.
In diesem Aufruf sollte überprüft werden, welches Ziel und welche Regeln gewählt wurden, um das Szenario anzupassen. Außerdem sollte die überladene Funktion per inherited() ausgeführt werden.

Syntax

public func ChooserFinished ()

Beispiel

public func ChooserFinished()
{
  //Spielziel HTF: Flaggenposten erstellen
  if (FindObject(GHTF))
    CreateObject(OFPL, 800, 300, NO_OWNER);  

  //Überladene Funktion, entfernt ggf. Munition und relauncht alle Spieler
  inherited();
}
