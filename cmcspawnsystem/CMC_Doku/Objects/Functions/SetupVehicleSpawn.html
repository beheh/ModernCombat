SetupVehicleSpawn

Kategorie: Funktionen

Beschreibung

Lässt ein Fahrzeug automatisch respawnen, wenn es zerstört wurde.

Script

bool SetupVehicleSpawn (array aType, int iDir, object pTarget, int iFrames, int iDistance);

Parameter

aType:
Array der möglichen Fahrzeug-IDs. Es wird immer eine zufällige ID ausgewählt.
[opt] iDir:
Richtung, in die die Fahrzeuge gedreht werden.
[opt] pTarget:
Spawnpoint, dem die Fahrzeuge hinzugefügt werden. 0 bei lokalem Aufruf.
[opt] iFrames:
Zeitintervall, nach dem ein Fahrzeug respawnt. Wenn nicht angegeben, respawnt ein Fahrzeug nach 2 Sekunden.
[opt] iDistance:
Distanz für den sicheren Bereich. Fahrzeuge außerhalb dieses Bereichs werden mit der Zeit zerstört. Wenn nicht angegeben, wird 100 angenommen.

Beispiel

var obj = CreateObject(VSPW, LandscapeWidth() / 2, 600, NO_OWNER);
SetupVehicleSpawn([PBOT], 0, obj);
Lässt Schlauchboote in der Mitte der Karte spawnen.
