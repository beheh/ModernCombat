AutoRespawn

Kategorie: Funktionen

Beschreibung

Lässt ein Objekt automatisch respawnen, wenn es zerstört oder getötet wurde.

Script

object AutoRespawn (int iFrames, int iDistance, object pTarget);

Parameter

[opt] iFrames:
Zeit, nach der ein Objekt respawnt wird. Wenn nicht angegeben, wird ein Objekt nach 30 Sekunden respawnt.
[opt] iDistance:
Wenn angegeben, wird ein Objekt bereits dann respawnt, wenn es mindestens so weit vom Spawnpoint entfernt ist.
[opt] pTarget:
Objekt, das respawnt wird. 0 bei lokalem Aufruf.

Beispiel

CreateObject(BIRD, LandscapeWidth() / 2, 200, NO_OWNER)->AutoRespawn(350);
Erzeugt einen Vogel, der bei seinem Tod respawnt.
