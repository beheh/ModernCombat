FlagLost

Kategorie: Ziele (Calls)

Beschreibung

Wenn ein Team einen Flaggenposten verliert, wird diese Funktion im Szenarienscript und in allen Zielen und Regeln aufgerufen.

Syntax

public func FlagLost (object pFlag, int iOldTeam, int iNewTeam, array aAttackers)

Parameter

pFlag:
Flagge, die eingenommen wurde.
iOldTeam:
Team, das die Flagge zuletzt hatte.
iNewTeam:
Team, das die Flagge nun besitzt.
aAttackers:
Array der Clonks, die die Flagge eingenommen haben.
