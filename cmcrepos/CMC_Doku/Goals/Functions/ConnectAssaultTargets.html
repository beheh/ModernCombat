ConnectAssaultTargets

Kategorie: Ziele (global)

Beschreibung

Verbindet mehrere Zielobjekte im Spielziel Assault. Verbundene Ziele werden gleichzeitig im Scoreboard angezeigt. Der Ticket-Reset findet erst statt, wenn alle verbundenen Ziele vernichtet wurden.

Syntax

bool ConnectAssaultTargets (array aTargets);

Parameter

aTargets:
Array der zu verbindenden Zielobjekt-Indizes, die jeweils bei AddAssaultTarget angegeben wurden.

Anmerkung

Es wird weiterhin für jedes Ziel einzeln OnAssaultTargetDestruction aufgerufen. Wird das letzte verbundene Ziel zerstört, ist jedoch der Parameter fConnectedDestruction true.

Beispiel

ConnectAssaultTargets( [2, 3, 4] );
Verbindet die Ziele mit den Indizes 2, 3 und 4.
