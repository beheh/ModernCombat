FlagCaptured

Kategorie: Ziele (Calls)

Beschreibung

Wenn ein Team einen Flaggenposten vollständig eingenommen hat, wird diese Funktion im Szenarienscript und in allen Zielen und Regeln aufgerufen.

Syntax

public func FlagCaptured (object pFlag, int iTeam, array aAttackers, bool fRegained)

Parameter

pFlag:
Flagge, die eingenommen wurde.
iTeam:
Team, das die Flagge nun besitzt.
aAttackers:
Array der Clonks, die die Flagge eingenommen haben.
fRegained:
Wurde die Flagge lediglich nach einem gescheiterten Angriff wieder auf 100% gebracht, ist dieser Parameter true.
