OnAssaultTargetDestruction

Kategorie: Ziele (Calls)

Beschreibung

Wird im Spielziel Assault bzw. Base Assault ein Zielobjekt zerstört, wird diese Funktion im Szenarienscript aufgerufen.

Syntax

public func OnAssaultTargetDestruction (object pTarget, int iTeam, int iIndex, bool fConnectedDestruction)

Parameter

pTarget:
Objekt, das zerstört wurde.
iTeam:
Teamnummer des Teams, dem das Objekt gehörte.
iIndex:
Index des zerstörten Ziels.
fConnectedDestruction:
Wenn das Objekt als letztes Zielobjekt eines Verbundes zerstört wurde, ist dieser Parameter true.

Anmerkung

Zum Zeitpunkt des Aufrufs existiert das Zielobjekt noch, es kann also im Aufruf vernichtet werden. Wird dies nicht getan, explodiert es standardmäßig mit der Stärke 50.

Beispiel

public func OnAssaultTargetDestruction(object pTarget)
{
  if (!Random(4)) CastObjects(ABOX, 5, 50, GetX(pTarget), GetY(pTarget));
}
Mit etwas Glück verschleudern zerstörte Zielobjekte Munition.
