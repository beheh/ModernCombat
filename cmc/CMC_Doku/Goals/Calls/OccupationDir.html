OccupationDir

Kategorie: Ziele (Calls)

Beschreibung

Wird von den Zielen Besitznahme und Base Assault im Szenarienscript aufgerufen, um festzustellen, in welche Richtung das Spielgeschehen verläuft.

Syntax

public func OccupationDir ()

Rückgabewerte

GOCC_Horizontal:
Spiel verläuft horizontal (zB Island Strike).
GOCC_Vertical:
Spiel verläuft vertikal (zB Babel Tower).

Anmerkung

Bei Rückgabewert 0 wird automatisch GOCC_Horizontal angenommen.
Die Richtung wird verwendet, um das Relaunch-Menü und ggf. das Scoreboard zu sortieren.
